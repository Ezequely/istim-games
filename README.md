+istim-games
 ============
  
Game manager API for the Istim Gaming Platform.
 
 
 
## Services
### Games
  - Publish game
  - Edit game info
  - Remove game
  - Search for available games
 
### Achievements
  - Create achievements for a game
  - Edit achievements info
  - Remove achievement
  - Search available achievements for a specific game
 
## Dependencies
### User API
  - Authentication
